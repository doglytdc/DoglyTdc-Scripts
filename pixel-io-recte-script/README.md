# Pixel iO Recte Script

Recte V3.2.4 para Poxel.io adaptado como bookmarklet (v5). Pipeline em 4 estágios: DoglyTdcSplash (tela de carregamento roxa com logo + glitch + nome do script por 2s) → bootstrap (intercepta addEventListener para capturar handlers do menu) → Recte.user.js (script original) → wrapper (botão flutuante com logo DoglyTdc + Alt+X). Toggle combina invocação direta dos handlers + dispatch sintético + foco no canvas. FAB se reparenta ao document.fullscreenElement automaticamente.
